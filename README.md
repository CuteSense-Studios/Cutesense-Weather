# 🌦️ Cutesense Weather & Mood Indicator

**Hand-crafted with love at Cutesense Studios.** A cozy, handwritten weather dashboard that translates atmospheric data into emotional moods and artistic vibes.

---

## ✨ Features

* **Mood-Based Forecasts:** Uses weather codes to determine your current "mood" (e.g., Joyful & Bouncy, Sleepy & Snuggly).
* **Atmospheric Effects:** Real-time visual layers including falling ink-style rain and drifting snowflakes based on your local weather.
* **Hand-Drawn Aesthetic:** A unique "paper-card" UI with custom ink-textured overlays, Kalam typography, and floating blob highlights.
* **Geolocation Aware:** Automatically detects your location via browser API with a fallback to IP-based detection.
* **Powered by Gemini:** Creative weather descriptions and logic woven with AI assistance.

## 🛠️ Technical Stack

* **Framework:** [React 19](https://react.dev/)
* **Build Tool:** [Vite 8](https://vite.dev/)
* **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
* **Icons:** [Lucide React](https://lucide.dev/)
* **API:** [Open-Meteo](https://open-meteo.com/) for weather data and [OpenStreetMap](https://www.openstreetmap.org/) for reverse geocoding.

## 🚀 Deployment

This project is deployed using **GitHub Actions**. Every push to the `main` branch automatically builds and deploys the site to GitHub Pages.

Check it out here: [https://Shubham-hahh.github.io/cutesense-weather/](https://Shubham-hahh.github.io/cutesense-weather/)
